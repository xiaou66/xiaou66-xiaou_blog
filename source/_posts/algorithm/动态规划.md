#  博弈类

## 模板

```java
private boolean canIWin(int n) {
    Boolean[] meno = new Boolean[n + 1];
    return dfs(n, meno);
}
private boolean dfs(int n, Boolean[] meno) {
    // index 边界线
    if(n < 0) {
        return false;
    }
    // meno 已有结果
    if(meno[n] != null) {
        return meno[n];
    }
    // 初始化结果
    boolean res = false;
    // 核心递推公式
    for(int i = 1; i < 4; i ++) {
        if(n >= i) {
            res |= !dfs(n - i, meno);
        }
    }
    // 返回结果并保存到 meno 中
    return meno[n] = res;
}
```
